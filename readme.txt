Cambios en el proyecto - Registro de versiones

Formato de registro:
[Fecha: YYYY-MM-DD]
- Cambios realizados:
- Nuevas funcionalidades agregadas:
- Elementos editados:
- Elementos eliminados:

Ejemplo:
[Fecha: 2024-06-01]
- Cambios realizados: Se corrigió el archivo package.json para ajustar versiones y nombre del paquete.
- Nuevas funcionalidades agregadas: Ninguna.
- Elementos editados: package.json
- Elementos eliminados: Ninguno.

Por favor, cada vez que se realice un cambio en la página, agregar una nueva entrada siguiendo este formato para mantener un control ordenado y cronológico.

[Fecha: 2024-06-06]
